# Referenzen — Offertenpruefung

Dieser Ordner enthaelt validierte Fachquellen und Vergleichsdaten mit **hoechster Prioritaet** fuer den Offertenpruefungs-Skill.

## Was gehoert hierhin?

### Branchen-Marktreports
- KBOB Stundensaetze (aktuell)
- Wuest Partner Marktberichte
- SIA-Honorartabellen (SIA 102/103/108)
- BFS Baupreisindex

### Vergleichsofferten (anonymisiert)
- Abgeschlossene Projekte mit verhandelten Reduktionen
- Marktvergleiche pro Gewerk

### Vertragsstandards
- SIA 118 (Allgemeine Bedingungen Bauarbeiten)
- SIA 1024 (Werkvertrag)
- KBOB-Vertragsvorlagen

### Buerointerne Lessons Learned
- Erfahrungswerte zu Anbietern
- Typische Reduktionen pro Gewerk
- Schwarze Liste (problematische Anbieter)

## Dateinamen-Konvention

`YYYYMMDD-Quelle-Thema.pdf` (gemaess JANS-Konvention)

z.B.:
- `20260301-KBOB-Stundensaetze-Bauhauptgewerbe.pdf`
- `20260315-WuestPartner-Marktbericht-Schweiz.pdf`
- `20260101-SIA-1024-Werkvertrag.pdf`
