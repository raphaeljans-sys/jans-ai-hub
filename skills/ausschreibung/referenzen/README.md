# Referenzen-Ordner — Skill `ausschreibung`

Hier liegen hochpriorisierte Fachquellen fuer den Skill `ausschreibung`. Sie werden bei jeder LV-Erstellung als Vorbild und Plausibilitaetscheck konsultiert.

## Was gehoert hierhin

- **Goldstandard-Beispiele**: Real eingegangene Offerten von Spezialplanern, die als Format-Vorbild dienen (z.B. Gastro-Online als Mustervorlage fuer schlanken Stil)
- **SIA-Normen**: SIA 118 (allgemeine Bedingungen), SIA 1024 (Submission-Verfahren) — soweit relevant
- **KBOB-Tarife**: Aktuelle KBOB-Honorartarife fuer Spezialplaner
- **Marktreports**: Wuest Partner, KBOB, Branchenstatistik mit Kostenkennwerten
- **AGB-Vorlagen**: SIA-118-konforme AGB, falls Submission formal abgesichert werden soll
- **Vergleichsofferten**: Anonymisierte Offerten aus abgeschlossenen Projekten als Benchmark

## Konvention

Dateinamen nach JANS-Rule: `JJJJMMTT-Titel.pdf` oder `JJMMTT-Titel.pdf`.

Beispiele:
- `260505-gastro-online-offerte-kispi-referenz.pdf`
- `2024-sia-1024-submission-spezialplaner.pdf`
- `2026-kbob-tarife-spezialplaner.pdf`

## Verknuepfung zu offertenpruefung

Der Skill `offertenpruefung` hat einen eigenen Referenzen-Ordner mit Branchenkennwerten. Wenn eine Referenz fuer beide relevant ist (z.B. KBOB-Tarife), liegt sie nur EINMAL im offertenpruefung-Ordner und wird hier per Symlink referenziert.
