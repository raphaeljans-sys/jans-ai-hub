# Glossar — Schweizer Bauleitungs-Fachbegriffe

Verbindliche Begriffe für Ausschreibung, Vergabe und Unternehmerkontrolle.
Quelle: BBase-Schulungsunterlagen (W693–W696) + JANS-Praxis.

| Begriff | Definition |
|---|---|
| **Abgebot** | Nachlass auf den Angebotspreis nach dem Offertvergleich; Verhandlungsergebnis mit dem Erstplatzierten. |
| **Vergabeeinheit (VE)** | Gliederung der Bauarbeiten in vergebbare Pakete (z.B. Gipser, Maler) — Basis für Ausschreibung und Unternehmerkontrolle; wird aus dem Kostenvoranschlag übernommen. |
| **Submittent** | Anbieter/Unternehmer, der eine Offerte einreicht. |
| **Submittentenliste** | Übersicht aller angefragten Unternehmer, gruppiert nach Vergabeeinheit/BKP. |
| **OVG (Offertvergleich)** | Strukturierte Gegenüberstellung der Angebote nach Qualität, Quantität, Einheitspreisen, Rabatten, Wirtschaftlichkeit und Terminen. |
| **Devis** | Leistungsverzeichnis / Angebotsformular (welsch für LV). |
| **Vergabeantrag** | Bewertung + Rankingempfehlung an den Bauherrn. |
| **Werkvertrag (WV)** | Vertrag mit dem ausgewählten Unternehmer (i.d.R. nach SIA 118). |
| **Akontorechnung** | Fortlaufende Teilzahlung des Unternehmers vor Projektabschluss, auf Basis Leistungsaufstellung. |
| **Ausmass** | Kontrolle/Aufnahme der tatsächlich erbrachten Mengen (Mengenermittlung). |
| **Nachtrag (NT)** | Zusatzleistung ausserhalb des Werkvertrags, mit schriftlicher Änderungsorder. |
| **Regie** | Stundenabrechnung bei unvorhersehbarer Mehrarbeit; Kostenblöcke Honorar/Dritte/Spesen. |
| **Solidarbürgschaft** | Garantie, bei der der Bürge für den gesamten Rechnungsbetrag haftet (Sicherheit für den Bauherrn). |
| **Schlussrechnung (SR)** | Endabrechnung nach Bauabschluss inkl. aller Nachträge, Abzüge, Regie — Gegenüberstellung zum KV. |
| **Abnahme** | Formale Entgegennahme des Werks; Mängelaufnahme mit Fristsetzung zur Behebung. |
| **Garantie / Gewährleistung** | Nach SIA 118 i.d.R. 2 Jahre Rügefrist + 5 Jahre verdeckte Mängel; Unternehmer behebt Mängel kostenlos. |
| **Garantieschein** | Bank- oder Versicherungsgarantie zur Sicherung der Gewährleistung. |
| **BKP (Baukostenplan)** | Schweizer Kostengliederung (CRB), Basis für KV und Baubuchhaltung. → `references/bkp-2017/`. |
| **NPK (Normpositionen-Katalog)** | Normierte Positionstexte (CRB) für Leistungsbeschriebe; Austausch via SIA 451. |
| **Kostenvoranschlag (KV)** | Detaillierte Kostenprognose vor Baubeginn, BKP-gegliedert, Toleranz ±10 % (SIA 102). |
| **Grobkosten** | Entwurfsbegleitende Kostenschätzung (SIA-Phasen 1–2), Kubik-/Flächenmethode + Kennwerte. |
| **Baubuchhaltung** | Laufende Gegenüberstellung Budget ↔ Verpflichtungen ↔ Zahlungen, BKP-/VE-gegliedert. |
| **Kostenarten H/D/S** | Honorar (eigene Leistung), Dritte (Fremdleistungen), Spesen (Kopien, Transport). |
| **Bauseits** | Leistungen, die NICHT im Auftrag des Unternehmers enthalten sind (Sanitär, Elektro, HLK …). |
| **Richtofferte** | Unverbindliche Vorab-Offerte zur Budget-/Plausibilitätskontrolle. |
| **«Gleichwertig»-Klausel** | «Gleichwertige Produkte zugelassen, sofern technische Spezifikationen vollumfänglich erfüllt» — juristisch wichtig bei öffentlichen Bauherren (IVöB/BöB). |
