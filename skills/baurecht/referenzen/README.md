(ergaenzend zur SharePoint-Gesetzessammlung und der lokalen Markdown-KB).

## Ablage-Regeln
- Dateiname: `JJJJMMTT-Titel-Kurzform.pdf` oder sprechender Name
- Nur validierte Fachquellen (Gesetze, Kommentare, Fachliteratur)
- Keine Entwuerfe oder Arbeitsdokumente
