# Referenzen — Skill `brandschutz`

Hochpriorisierte Referenzdokumente fuer den Brandschutz-Skill. Jeder Eintrag mit Datum, Titel und Kurzbeschreibung der Verwendung.

## Goldstandards

### 260513-Goldstandard-Factsheet-Ertuechtigung-KISPI.pdf
Erstes vollstaendiges Brandschutz-Ertuechtigungs-Factsheet (KISPI Therapiestation 1.OG, 13. Mai 2026). Definiert den Goldstandard fuer:
- 10-Kapitel-Struktur (Zweck, Anforderungen, Ist-Zustand, Wandgruppen-Abgleich, Massnahmen, Material, Nachweispflichten, Schnittstellen, LV-Klauseln, naechste Schritte)
- Tabellen-Layout im JANS-Dokument-Standard
- Verbindliche LV-Klauseln A bis E fuer die Ausschreibung
- Methodik Wandgruppen-Abgleich Brandschutzplan vs. Architekturplan

Verwendung als Vorlage fuer:
- KISPI weitere Etagen / Sektoren
- andere Healthcare-Umnutzungen mit Bestandeswaenden
- generell Brandschutz-Ertuechtigung Bestand bei Schweizer VKF-Projekten

## Praxisfaelle

Dokumentierte Einzelfaelle als Wissensbasis. Ordner: `praxisfaelle/`.

### 260520-KISPI-Bolon-Sisal-Bodenbelag-Fluchtweg
Bereits verlegter Bodenbelag Bolon Sisal Plain Sand BKB (Bfl-s1) im Fluchtweg Neubau Kinderspital Zuerich. Goldstandard fuer das Muster "Nachweis statt Neuausschreibung":
- Unterlagen-Beschaffung beim Lieferanten (Rechnung, DoP, Datenblatt)
- Brandverhaltens-Klassifizierung extrahieren (EN 13501-1)
- Beauftragung Brandschutz-Fachplaner (Gruner) zur Pruefung mit Feuerpolizei
- Schriftliche Bestaetigung an Bauherrschaft (KISPI) anstoßen
