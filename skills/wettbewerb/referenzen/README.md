# Referenzen — Skill `wettbewerb`

Hochprioritaere Fachquellen fuer den Wettbewerbs-Harness. PDFs mit dem Namensschema
`JJJJMMTT-Titel.pdf` ablegen und hier im Inhaltsverzeichnis vermerken.

## Was gehoert hierhin
- **SIA 416** (Flaechen/Volumen von Gebaeuden) — Bezugsgroessen fuer den Flaechen-Nachweis
- **SIA 142/143** (Ordnung fuer Architektur-/Planungswettbewerbe und Studienauftraege) —
  Verfahrens-/Abgaberegeln, Anonymitaet, Preisgericht
- **Goldstandard-Wettbewerbsprogramme** (anonymisiert) als Muster fuer `programm-leser`
- **Goldstandard-Abgaben JANS** (Boards, Erlaeuterungsberichte) als Layout-/Textmuster
