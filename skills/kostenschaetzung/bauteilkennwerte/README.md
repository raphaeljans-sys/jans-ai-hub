# Bauteilkennwerte — Kostenschaetzung

Hier liegen Bauteil-/Positions-Kennwerte als Ergaenzung zu den m2-NF-Kennwerten
im `referenzen/`-Ordner. Waehrend `referenzen/` die m2-Bezugsgroessen (BKP 1-5
pro m2 NF) enthaelt, sind hier konkrete Bauteile mit Stueck-, Lauf- oder
m2-Bauteil-Preisen abgelegt — fuer detailliertere Schaetzungen im Phase 31/41
(Vorprojekt / Bauprojekt) oder bei Einzelpositions-Anfragen.

## Ablage-Regeln
- Dateiname: `JJJJMMTT-Bauteil-Kurzform.md` (oder `.pdf` bei externen Quellen)
- Markdown fuer buerointerne Marktorientierung / Recherche
- PDF fuer validierte Submissions-Auswertungen / externe Quellen
- Quelle, Stand, Bandbreite und Annahmen IMMER mit angeben
- Werte sind Marktorientierungen — KEINE Submissionswerte
