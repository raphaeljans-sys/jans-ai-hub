# JANS Templates

Vorlagen fuer Dokumente, Reports und Outputs.
Werden von Agents und Plugins verwendet.

## Verwendung

Templates werden von Agents referenziert, z.B.:
- Baurechts-Report-Vorlage
- Offert-Vorlage
- Protokoll-Vorlage

Neue Templates hier ablegen und in der jeweiligen Agent-Definition referenzieren.
