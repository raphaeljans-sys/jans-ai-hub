# Recht & Norm — Quellenlandkarte (PL-02)

**Kein Doppel** zur Baurechts-Wissensbasis: rechtliche Wuerdigung laeuft ueber den Skill
`baurecht` und die KB `wissen/baurecht`. Dieser Artikel ist nur der **Wegweiser**, wo in
**PL - 02_Recht_Norm** welche Rohquelle liegt — damit die Grundlagenbeschaffung schnell die
richtige Originalquelle zitiert.

## Struktur PL-02 (Arbeitsstand)

| Unterordner | Inhalt |
|---|---|
| `01_Gesetze/01_Bund` | RPG (700er), USG/LRV (814er), EnG/EnV (730er), BehiG, Eisenbahn-/Zivilschutz |
| `01_Gesetze/02_Schwyz` | PBG/PBV SZ, Ausnuetzung (Art. 17/31i), Waldabstand (Art. 35), Gebaeude-/Firsthoehe |
| `01_Gesetze/02_Zuerich` | Planung, Energie, Submission, Wasser, Zivilschutz |
| `01_Gesetze/03_Rechtsprechung` | VGer ZH, BGE-Auszuege |
| `01_Gesetze/05_Dispensrecht` | Dispens-/Ausnahmebewilligungen (BRKE) |
| `01_Gesetze/08_Baurechtsanalysen` | reale JANS-BRA (Thalwil, Maur, Zuerich, Willerzell, Waedenswil…) |
| `02_Normen/SIA_Norm` | SIA Normen/Merkblaetter/Vertraege |
| `02_Normen/VKF_Norm` | Brandschutznorm 2015 → siehe Skill `brandschutz` / PL-03 |
| `02_Normen/DIN_Norm`, `RAL_Norm`, `Lignum` | weitere Normwerke |

## Nutzung
- Rechtsfrage → Skill `baurecht` (zitiert aus dieser Sammlung + SharePoint-Gesetzessammlung).
- Reale Praezedenz gesucht → `08_Baurechtsanalysen` (anonymisierbar als Benchmark).
- Normtext gesucht → `02_Normen/<Normgeber>`.
