## Kernbefunde

**FEA-Klassengrenzen (Uw,eq in W/(m²·K)):**

| Klasse | Bereich | | Klasse | Bereich |
|---|---|---|---|---|
| A | < 0 (Energiegewinn) | | E | ≥ 0,3 bis < 0,4 |
| B | ≥ 0 bis < 0,1 | | F | ≥ 0,4 bis < 0,8 |
| C | ≥ 0,1 bis < 0,2 | | G | ≥ 0,8 |
| D | ≥ 0,2 bis < 0,3 | | | |

Musterbeispiel: Uw 1,1 / Ug 0,6 / Uf 1,1 / g-Wert 60 % / ψg 0,03 → Uw,eq = −0,08 → **Klasse A**. Zeigt
konkret: ein solider (nicht rekordtiefer) Uw kombiniert mit hohem g-Wert reicht für die Bestklasse —
ein Verkaufsargument gegen «immer den tiefsten Uw kaufen».

**GEAK-Kostenspanne (Marktrichtwerte, Sekundärquelle):**

| Gebäudetyp | GEAK | GEAK Plus |
|---|---|---|
| EFH | CHF 450–650 | CHF 1'400–2'100 |
| MFH | CHF 500–800 | grössenabhängig, keine feste Spanne |

Plus amtliche Zertifizierungsgebühren CHF 80 (GEAK) / 190 (GEAK Plus). Bei einem EFH-GEAK-Plus deckt
die kantonale Förderung (ZH/SZ CHF 1'500) damit oft mehr als die Hälfte der Kosten.
