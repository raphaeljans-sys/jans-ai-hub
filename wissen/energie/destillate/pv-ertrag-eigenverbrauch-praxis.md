# PV-Ertrag & Eigenverbrauch — zwei Praxisbeispiele (kWh/kWp)

**Das Wichtigste in 1 Satz:** Eine PV-Anlage im Schweizer Mittelland/Voralpenraum liefert real
rund **950–1'025 kWh pro kWp und Jahr**, produziert im Sommer aber **3–4× so viel wie im Winter**,
und **wie viel davon der Bauherr selbst nutzt (Eigenverbrauch/Autarkie) haengt fast nur am
Lastprofil** — von ~8 % (1-Personen-Haushalt) bis ~44 % (grosser Haushalt) ohne Batteriespeicher.

## Beispiel A — SolarApp, Reckholdern 20, 8846 Einsiedeln (Willerzell), V1 09.01.2024
Indach-Anlage (MATCH tile Indachsystem, Glas-Glas-Module 25/32/76 Wp; Growatt MOD 6000TL3-X, 6 kW,
2 MPPT). Vorprojekt mit Auslegung + Nutzenanalyse.

| Kennzahl | Wert |
|---|---|
| Installierte Leistung | **7,90 kWp** (252 Module, Fläche 92 m²) |
| Relative Produktion | **981 kWh/kWp** |
| Stromproduktion | **7'743 kWh/a** |
| Stromverbrauch Haushalt | 14'500 kWh/a (gross) |
| **Eigenverbrauch** | 3'386 kWh = **44 %** der Produktion |
| **Unabhaengigkeit (Autarkie)** | **53 %** |
| Einspeisung ins Netz | 4'356 kWh/a |
| Investition (inkl. MwSt) | CHF 58'198 (Material 42'218 + Arbeit 11'619) |
| Foerderung/Steuervorteile | CHF 3'674 |
| Ertrag pro Jahr | CHF 1'172 |
| CO₂-Reduktion | 3'391 kg/a |
| Energetische Amortisation (graue Energie) | **2 Jahre** |

**Saisonalitaet (Monatsproduktion kWh):** Jan 351 · Feb 462 · Mrz 656 · Apr 819 · Mai 820 ·
Jun 840 · Jul 896 · Aug 829 · Sep 745 · Okt 639 · Nov 385 · Dez 299 → Sommer ≈ **3× Winter**.
Monatliche **Eigenverbrauchsquote** 37 % (Jul) bis 61 % (Dez); **Autarkie** 21 % (Dez) bis 89 % (Jul).
Tagesgang: Sommer-Produktion bis ~3,4 kW, Winter bis ~1,8 kW (Mittag-Peak, morgens/abends Netzbezug).
Hinweis im Report: Produktionswerte aus **PVGIS** (Ausrichtung/Neigung/Verschattung Berge);
Eigenverbrauch aus Erfahrungswerten des gewaehlten Verbraucherprofils.

## Bauherren-Transfer (eigene Einordnung markiert)
- **Spezifischer Ertrag** als Planungs-Faustwert CH Mittelland/Voralpen: **~950–1'025 kWh/kWp·a**
  (beide Beispiele real; gut besonntes, sued-/leicht-ostorientiertes Dach mit ~26° Neigung). Steiler
  Nord-/Schattenlage tiefer, Flachdach Sued-Aufstaenderung aehnlich. → Schnellschaetzung Jahresertrag
  = **kWp × ~1'000 kWh**.
- **Eigenverbrauchsquote OHNE Batterie ist fast reine Lastprofil-Frage:** Single ~8 % (B), grosser
  Haushalt mit WP/E-Auto ~44 % (A). **Wer den Eigenverbrauch heben will:** Anlage aufs eigene
  Verbrauchsprofil dimensionieren (nicht aufs Dach maximieren), **WP/Warmwasser/E-Auto tagsueber
  laden** (Lastverschiebung), erst dann Batteriespeicher. Autarkie folgt dem Eigenverbrauch.
- **Saisonalitaet** ist hart (Sommer 3–4× Winter): PV deckt den Winter-Heizstrom einer WP **nicht**
  ab — die Jahresbilanz «Plusenergie» heisst Netzaustausch, nicht Inselbetrieb.
- **Flaeche/kWp:** Aufdach-Vollmodul ~5–8 m²/kWp (B: 32 m²/6 kWp ≈ 5,3); **Indach-Ziegelsystem
  braucht mehr Flaeche** (A: 92 m²/7,9 kWp ≈ 11,6 m²/kWp wegen kleinteiliger Solarziegel + Randplatten).
- **Wirtschaftlichkeit:** Nach Foerderung (Einmalverguetung Bund + Steuerabzug, vgl. `[[foerderprogramm-energie-zh-2026]]`)
  Amortisation grob 12–24 J je nach Eigenverbrauch/Strompreis; energetische Amortisation der grauen
  Energie nur ~2 Jahre.

## Offene Punkte
- Beide Reports sind **Vorprojekt-/Tool-Schaetzungen** (PVGIS/Meteonorm), keine gemessenen Jahres-
  ertraege; CHF-Werte 2023/2024 marktabhaengig. Fuer ein Projekt gilt die Standortabfrage
  (PVGIS/sonnendach.ch) + Installateur-Offerte.
- Ertrags-Kennzahlen je **Ausrichtung/Neigung** nur ueber diese zwei Punkte belegt — systematische
  Tabelle (Ost/Sued/West × 0/15/30/45°) noch offen.
